# **App Name**: CricketConnect

## Core Features:

- Match Creation: Create new matches with customizable teams, players, and overs.
- Team Management: Add, edit, and rename teams and players with no player limit.
- Live Scorecard: Display real-time scores, current over stats, and batting/bowling statistics.
- Player Selection: Interactive dialog boxes to select the next batsman or bowler.
- Innings Management: Automatic transition to the next innings after completion of the current one.
- Live Sharing: Share live match scores via a generated link or QR code.

## Style Guidelines:

- Primary color: A vibrant and sporty blue (#4285F4) to represent energy and focus.
- Background color: Light gray (#F5F5F5), providing a clean and modern backdrop.
- Accent color: A lively green (#34A853) for key actions and highlights, sitting near blue on the color wheel and providing contrast.
- Headline font: 'Poppins', a geometric sans-serif font, for headlines to give a precise and contemporary look.
- Body font: 'PT Sans', a humanist sans-serif, providing readability for in-depth stats and information.
- Use clear, universally recognizable icons for actions like adding players, starting matches, and sharing.
- Use subtle transitions and animations when updating scores and player statistics.
